# 基于uniapp开发一个模仿网易云音乐的小程序
## 项目目录结构
- common 公用文件
- component 自定义组件目录
- pages 页面存放目录
- static 静态资源目录
- unpackage 编译后的文件存放目录
- App.vue 
- main.js
- manifest 项目配置
- pages.json 页面配置
- README.md 项目描述
- uni.scss 全局样式
